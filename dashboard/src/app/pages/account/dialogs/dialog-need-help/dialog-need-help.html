Verifying using Authenticator App

Scan the QR Code

  1. Open the authenticator app on your phone. If you don’t have it yet, follow the instructions below to download it first.
  2. In the app, look for an option to add a new account. This could say "Scan QR code," "Add Account," or something similar.
  3. Scan the QR code shown on your computer screen using your phone’s camera (within the app).
  4. The app will generate a 6-digit code.

Enter the Code

  1. Enter the 6-digit code from the authenticator app into the box on your computer screen.
  2. Click "Submit" to continue.
  3. If the code is correct, you’ll be logged in to your account.

What to Do if You Have Trouble Scanning or Entering the Code

  1. Make sure the QR code is clear: Try adjusting your phone's camera or moving it closer to the screen if the code doesn’t scan.
  2. If the code doesn’t work: Double-check that you entered all 6 digits correctly and that you’re using the most recent code from the app (they change every 30 seconds).
